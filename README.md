# 弾幕王コウカトン（仮）

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 主人公コウカトンが強大な敵に立ち向かうゲーム

## ゲームの遊び方
* 強大な敵をコウカトンの弾幕で倒し切る
* 強大な敵の様々なギミックを搔い潜ってボスを倒せ

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画
* コウカトンの動作
* ボス実装

### 分担追加機能
* ボスとコウカトンのHP   :c0b23021
* ボスの攻撃 :(仮)
* コウカトンの攻撃,MPゲージ :c0a23135
* ボスの攻撃反射 :c0b23161
